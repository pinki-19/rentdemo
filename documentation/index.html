Folder Structure

  • Thebootstrap folder - contains all the Bootstrap that are used in html.
  • The images folder - contains all the images that are used in html and stylesheets for the layout purpose & it has subfolder named layerslide-gallery
  • The fonts folder - contains all the fonts that are used in html,
  • Theless folder - contains all the Less that are used in html
  • The scripts folder - contains javascript files & plugins ( Which includes)
  • TheStyles folder - contains all the Styles that are used in html

CSS Structure

The style.css is the main css file used in this template. This file also contains some general styling, such as anchor tag colors, font-sizes, etc., Note, that these properties might be overridden somewhere else in the file.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
